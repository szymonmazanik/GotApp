# GotApp
## Example MVVM Android app in Kotlin with pagination
### Reads data from api and presents them in list view.
### Tools: 
* Architecture Components: 
	- Data Binding
	- LiveData
	- ViewModel
	- Navigation
	- Paging
* RxJava, RxKotlin, RxAndroid
* Retrofit
* Moshi
* Timber
* JUnit, Espresso
